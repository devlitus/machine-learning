# Actividad 4: Clasificación Multiclase con Scikit Learn y TensorFlow

## Objetivo
Predecir la calidad del corte (`cut`) de un diamante utilizando modelos de clasificación multiclase sobre el dataset Diamonds de Seaborn.

## Dataset
- **Fuente**: Seaborn Diamonds dataset
- **Registros**: 53,940 diamantes
- **Variables objetivo**: 5 categorías de corte (Ideal, Premium, Very Good, Good, Fair)
- **Features**: carat, color, clarity, depth, table, price, x, y, z

## Contenido
- Carga y análisis exploratorio de datos
- Limpieza de datos (valores nulos, duplicados, outliers)
- Preprocesamiento con Scikit-Learn
- Modelos de clasificación:
  - Random Forest Classifier
  - Logistic Regression
  - Support Vector Machine
  - Decision Tree Classifier
  - Red neuronal con TensorFlow/Keras

## Ejecución
Ejecutar el notebook `diamonds_clasificacion_multiclase.ipynb` para ver el análisis completo y resultados de los modelos.
